How the tournament works

The tournament normally lasts 3 rounds. Each round, every contestant makes one Wisdom (Survival) fishing check. The final score is the total number of Catch Points earned across all rounds.

  • 5 or lower: Bad Catch table
  • 6–10: No Catch
  • 11–14: Minor Catch table
  • 15–19: Good Catch table
  • 20–24: Great Catch table
  • 25+: Master Catch table

If a catch asks for a saving throw, the save DC equals the selected fishing ground DC.

Cheating rule

Before the fishing check, a contestant may cheat using Dexterity (Sleight of Hand) or Charisma (Deception) against DC 15. On success, add +5 to that round's fishing check. On failure, the contestant is caught and disqualified. Classic tavern sportsmanship: everybody smiles until the fake carp comes out of someone’s coat.
